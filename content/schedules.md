# schedules list definition


The schedules list specifies the scheduled triggers for the pipeline.


:::moniker range="= azure-pipelines-2020"

<!-- :::api-definition signature="schedules[schedule]" version="azure-pipelines-2020"::: -->

```yaml
schedules: [ cron ] # 
```
